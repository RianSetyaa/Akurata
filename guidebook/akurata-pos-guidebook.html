1. Gambaran umum

Akurata POS adalah sistem backoffice dan kasir berbasis web untuk membantu bisnis mengelola transaksi harian, stok, pembelian, piutang, laporan, dokumen PDF, dan sinkronisasi Loyverse.

Kasir

Transaksi dapat berisi lebih dari satu produk, diskon per item, pajak otomatis, pembayaran tunai, QRIS, transfer, atau tempo.

Stok

Produk mendukung satuan, volume/berat, barcode, kamera scanner, import Excel, dan composite item untuk paket.

Integrasi

Produk dapat dipush ke Loyverse, receipt Loyverse dapat ditarik, dan pajak/store Loyverse dapat disinkronkan.

Role

2. Peran pengguna dan batas akses

Administrator

Mengelola semua outlet, melihat akun, menghapus akun/toko secara aman, menangani chat CS, dan impersonate ke outlet tertentu.

Owner

Pemilik bisnis. Dapat membuka semua fitur outlet: transaksi, produk, PO, piutang, laporan, integrasi, log, profil bisnis, dan akses tim.

Manager

Pengguna backoffice dengan fitur dibatasi. Tidak dapat membuka Integrasi Loyverse, Log, dan edit Profil Bisnis.

Prinsip akses: pengguna hanya boleh melihat data outlet aktifnya. Administrator dapat masuk ke outlet via impersonate, tetapi aksi impersonate tidak perlu dicatat sebagai log outlet.

Masuk akun

3. Login, daftar, OTP, dan lupa password

Halaman login digunakan untuk membuka dashboard berdasarkan sesi akun.

Login

  1. Buka halaman login Akurata POS.
  2. Masukkan email atau nomor WhatsApp yang terdaftar.
  3. Masukkan password.
  4. Aktifkan Ingat saya jika ingin sesi bertahan lebih lama.
  5. Klik Masuk.

Daftar dan OTP

  1. Buka halaman daftar Akurata POS.
  2. Isi nama bisnis, nama pengguna, email/WhatsApp, dan password.
  3. Sistem mengirim kode verifikasi ke email.
  4. Masukkan kode verifikasi sebelum kedaluwarsa.
  5. Setelah valid, akun dan outlet pertama dibuat.
Jika kode verifikasi tidak masuk, cek folder spam/promosi, pastikan email benar, lalu ulangi pendaftaran atau hubungi tim support Akurata.
Operasional

4. Dashboard operasional

Dashboard menampilkan omzet, transaksi, margin, stok kritis, aktivitas kasir, dan shortcut restock.

  1. Gunakan kartu metrik untuk melihat kondisi harian secara cepat.
  2. Cek grafik per jam untuk membaca jam ramai dan pola transaksi.
  3. Gunakan daftar stok kritis untuk membuat PO cepat.
  4. Gunakan transaksi terbaru untuk membuka detail dan preview dokumen.

Jika data belum tampil, tunggu beberapa saat lalu refresh halaman. Jika masih belum muncul, hubungi admin atau tim support.

Penjualan

5. Penjualan, pajak, diskon, dan invoice

Transaksi baru dapat berisi beberapa item, diskon, pajak, dan pilihan metode pembayaran.

  1. Buka menu Penjualan.
  2. Klik Transaksi baru.
  3. Pilih metode pembayaran: Tunai, QRIS, Transfer, atau Tempo.
  4. Pilih produk, qty, dan diskon jika ada.
  5. Jika pajak diaktifkan, sistem menghitung subtotal + pajak.
  6. Klik Simpan transaksi.
  7. Buka Preview untuk melihat invoice PDF.
Pajak dicatat sebagai kolom terpisah. Di laporan penjualan, pajak tidak dihitung sebagai keuntungan.

Quotation

Quotation digunakan untuk membuat penawaran sebelum transaksi final. Owner dapat menampilkan atau menyembunyikan menu Quotation melalui Profil Bisnis. Jika fitur dimatikan, halaman akan memberi tahu bahwa Quotation tidak sedang ditampilkan.

Produk

6. Produk, stok, barcode, import Excel, dan paket

Tabel produk mendukung filter, pagination 10-100 produk, dan dropdown aksi agar tampilan tetap ringkas.

Tambah atau edit produk

  1. Klik Tambah produk atau pilih Edit produk dari dropdown aksi.
  2. Isi SKU, barcode, nama produk, kategori, tipe jual, stok, HPP, dan harga jual.
  3. Pilih Volume / Berat untuk produk desimal seperti beras, kopi kiloan, atau cairan.
  4. Pilih Composite / Paket untuk produk paket seperti rokok per bungkus berisi beberapa batang.

Import Excel dan barcode

  1. Klik Unduh template untuk mengambil format Excel.
  2. Isi sheet Produk dan Komponen.
  3. Upload file melalui Import Excel.
  4. Barcode dapat diketik manual atau dipindai menggunakan kamera.
Agar push produk ke Loyverse tidak gagal, pastikan SKU maksimal 40 karakter, nama maksimal 64 karakter, barcode unik, harga tidak negatif, dan produk volume hanya dipakai jika qty boleh desimal.
Pembelian

7. Purchase Order dan terima barang

PO mendukung lebih dari satu barang dan tidak langsung menambah stok sampai barang diterima.

  1. Buka menu Purchase Order.
  2. Klik Buat purchase order.
  3. Isi supplier dan beberapa item barang.
  4. Simpan PO. Status masih Menunggu.
  5. Saat barang datang, klik Terima barang.
  6. Masukkan biaya tambahan seperti ongkir, handling, atau biaya lain.
  7. Setelah diterima, stok produk bertambah dan HPP disesuaikan.
Rumus HPP PO: Pembelian per pcs + (Biaya lain / Qty). HPP baru disesuaikan dengan pendekatan rata-rata: HPP PO dan HPP sebelumnya dirata-ratakan.
Piutang

8. Piutang, DP, pembayaran parsial, dan invoice pembayaran

Piutang muncul dari transaksi tempo dan dapat dibayar sebagian sampai lunas.

  1. Pada transaksi penjualan, pilih metode Tempo.
  2. Isi pelanggan, jatuh tempo, dan DP jika ada.
  3. Buka menu Piutang untuk melihat tagihan.
  4. Klik Bayar untuk mencatat pembayaran penuh atau sebagian.
  5. Setiap pembayaran dapat dibuatkan invoice pembayaran PDF.
  6. Edit jatuh tempo jika kesepakatan pembayaran berubah.
Jika transaksi tempo belum muncul di Piutang, refresh halaman dan pastikan metode pembayaran transaksi benar-benar menggunakan Tempo. Jika masih belum muncul, hubungi admin atau tim support.
Laporan

9. Laporan penjualan dan pembelian

Laporan memisahkan subtotal, diskon, pajak, total, HPP, dan laba kotor.

Laporan penjualan

  1. Buka menu Laporan Penjualan.
  2. Gunakan pencarian untuk menemukan invoice, pelanggan, atau kasir.
  3. Perhatikan kolom pajak karena pajak bukan laba.
  4. Gunakan angka laba kotor untuk evaluasi produk dan margin.

Laporan pembelian

  1. Buka menu Laporan Pembelian.
  2. Cek PO, supplier, tanggal, total, dan biaya tambahan.
  3. Gunakan data ini untuk audit HPP dan stok masuk.
Pengaturan

10. Profil bisnis, pajak, rekening, dan logo dokumen

Profil Bisnis mengatur identitas outlet, logo dokumen, rekening penagihan, pajak, dan fitur Quotation.

  1. Buka Pengaturan > Profil Bisnis.
  2. Isi nama usaha dan alamat yang akan muncul pada dokumen.
  3. Upload logo usaha. Logo digunakan pada invoice, quotation, dan dokumen pembayaran.
  4. Isi rekening penagihan untuk invoice tempo.
  5. Aktifkan atau nonaktifkan pajak invoice dan isi persentase pajak.
  6. Aktifkan atau nonaktifkan fitur Quotation.
Saat pajak dinyalakan dan integrasi Loyverse aktif, sistem dapat membuat atau menyinkronkan pajak ke Loyverse.
Loyverse

11. Integrasi Loyverse

Integrasi Loyverse menyimpan akun merchant, POS/store aktif, sinkron produk, pajak, stok, dan receipt.

  1. Buka menu Integrasi.
  2. Klik Connect Loyverse jika belum terhubung.
  3. Login dengan akun Loyverse dan berikan izin akses yang diminta.
  4. Pilih POS/store yang dipakai outlet.
  5. Klik Push produk untuk mengirim produk Akurata ke Loyverse.
  6. Klik Tarik penjualan untuk membaca receipt Loyverse dan membuat invoice di Akurata.
Pastikan akun Loyverse yang dipakai adalah akun bisnis yang benar dan memiliki akses untuk mengelola produk, stok, pajak, POS, dan transaksi.
Akses dan CS

12. Akses user, log aktivitas, dan chat CS

Owner dapat mengelola user outlet. Chat CS membantu outlet berkomunikasi dengan administrator tanpa refresh manual.

Akses user

  1. Buka Pengaturan > Akses.
  2. Tambahkan Manager atau Cashier sesuai kebutuhan.
  3. Gunakan ubah password jika akun tim perlu direset.
  4. Gunakan hapus akun untuk menonaktifkan user tanpa merusak histori transaksi.

Log dan chat

  1. Log aktivitas mencatat login, logout, transaksi, produk, PO, integrasi, dan aktivitas penting lain.
  2. Log memakai pagination agar tidak memuat terlalu banyak data sekaligus.
  3. Chat CS mendukung sesi aktif, riwayat, dan sesi baru setelah chat berakhir.
Administrator

13. Laman Administrator

Administrator mengelola outlet, akun, integrasi, dan chat lintas tenant.

  1. Login menggunakan akun role administrator.
  2. Buka laman Administrator dari menu yang tersedia.
  3. Cari outlet atau akun yang ingin ditinjau.
  4. Gunakan Impersonate untuk masuk sebagai outlet yang dituju.
  5. Gunakan hapus akun atau toko jika diperlukan sesuai SOP internal.
Hapus toko dan akun adalah tindakan sensitif. Pastikan tindakan ini sudah disetujui pemilik bisnis dan tidak sedang ada pekerjaan operasional yang berjalan.
Mobile

14. Penggunaan HP/mobile

Tampilan mobile menyusun tabel menjadi kartu agar tetap mudah dibaca di layar kecil.

  1. Gunakan menu mobile untuk membuka halaman dashboard.
  2. Form transaksi, PO, produk, dan piutang akan berubah menjadi satu kolom.
  3. Tabel berubah menjadi kartu dengan label kolom di setiap baris.
  4. Scan barcode kamera memerlukan izin kamera dari browser. Gunakan koneksi web yang aman agar izin kamera dapat berjalan normal.

16. Checklist sebelum mulai menggunakan Akurata

  1. Login sebagai Owner dan lengkapi Profil Bisnis.
  2. Upload logo usaha agar tampil pada invoice dan dokumen.
  3. Isi rekening penagihan jika bisnis memakai pembayaran tempo.
  4. Atur pajak sesuai kebutuhan bisnis.
  5. Tambahkan user Manager atau Cashier sesuai tugas tim.
  6. Masukkan produk awal atau gunakan Import Excel.
  7. Atur barcode dan tipe produk satuan, volume, atau composite jika diperlukan.
  8. Hubungkan Loyverse jika bisnis memakai POS Loyverse.
  9. Uji satu transaksi kecil dan cek invoice PDF.
  10. Cek laporan penjualan dan stok setelah transaksi tersimpan.

Akhir dokumen. Simpan PDF ini sebagai panduan penggunaan untuk tim outlet dan perbarui jika fitur baru ditambahkan.

Catatan pembaruan: Navigasi dashboard kini menggunakan baris menu horizontal di bagian atas halaman, menggantikan sidebar kiri. Sub-menu (Penjualan, Laporan, Pengaturan) dapat dibuka melalui dropdown pada menu yang bersangkutan. Pada tampilan mobile, navigasi dapat diakses melalui tombol hamburger di sebelah kanan bar navigasi.
